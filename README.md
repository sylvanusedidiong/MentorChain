# MentorChain

A decentralized mentorship platform connecting experienced professionals with mentees for career development on Stacks blockchain.

## Features

- Mentor profile registration with expertise areas and experience
- Session-based mentorship booking with fee payments
- Program coordinator certification for mentor quality assurance
- Session tracking and mentorship relationship management
- Professional development through blockchain-verified mentoring

## Smart Contract Functions

### Public Functions
- `register-mentor` - Register as a mentor with expertise and background
- `book-mentorship` - Book a mentorship session with payment
- `certify-mentor` - Certify mentor credentials (coordinator only)

### Read-Only Functions
- `get-mentor` - Get mentor profile details
- `get-session-record` - Get specific session information
- `get-session-count` - Get total sessions for mentor
- `get-program-stats` - Get program statistics

## Usage

Deploy the contract to create a decentralized mentorship platform where experienced professionals can guide others in their career development through paid mentoring sessions.
